Simple expression

Write a program that reads four large integers a, b, c, d and produces the result of the expression below:

(-a) * b + c - d

Sample Input:
-9999999999999999
10000000000000000
20000000000000000
9999999999999999

Sample Output:
100000000000000000000000000000001



Memory limit: 256 MB
Time limit: 8 seconds
