Arithmetic expression

Write a program that reads 2 (two) numbers and prints their sum. The examples below demonstrate how the result should look like.

Input: two numbers of type Int.

Output: single string. Use the following template:

"n1 plus n2 equals sum", where n1 and n2 — input numbers, and sum — their sum.


Sample Input:
12
34

Sample Output:
12 plus 34 equals 46


Sample Input:
1
234

Sample Output:
1 plus 234 equals 235



Memory limit: 256 MB
Time limit: 8 seconds
