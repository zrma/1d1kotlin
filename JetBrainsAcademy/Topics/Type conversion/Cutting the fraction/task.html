Cutting the fraction

Write a program that reads a number and cuts its fractional part. As the result must be printed the integer part of the input number.

Use Double as the input type and Long as the result type, as a value can be quite large.

Input: single number of type Double.

Output: single number of type Long.


Sample Input:
5.44

Sample Output:
5



Memory limit: 256 MB
Time limit: 8 seconds
