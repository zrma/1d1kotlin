Standard output

Standard output is a place that a program can send some information to. It is supported by all common operating systems, but note that not all programs generate such output. By default, the standard output displays data on the screen, but it is possible to redirect it to a file.

During this course, you will often write programs that send some data (for example, strings and numbers) to the standard output.

Printing text

Kotlin has two functions to send data to the standard output: println and print.

The function println (print line) displays the passed string followed by a new line on the screen. For example, the code fragment below prints four lines:

println("I")
println("know")
println("Kotlin")
println("well.")

Output:

I
know
Kotlin
well.

As you can see, all the strings were printed as they are, without double-quotes.

You can also print an empty line when there is no string to print:

println("Kotlin is a modern programming language.")
println() // prints an empty line
println("It is used all over the world!")

Output:

Kotlin is a modern programming language.

It is used all over the world!

The function print displays the passed value and places the cursor (the place where to print) after it. Let's look at the example below. This piece of code outputs all strings in a single line.

print("I ")
print("know ")
print("Kotlin ")
print("well.")

Output:

I know Kotlin well.

Now, you can write "Happy New Year" to your friends, using Kotlin!

Printing numbers and characters

The functions println and print allow a program to print not only strings but also numbers and characters.

Let's print two secret codes.

print(108)   // prints a number
print('c')   // prints a character
print("Q")   // prints a string
println('3') // prints a character that represents a digit 

print(22)
print('E')
print(8)
println('1')

Output:

108cQ3
22E81

Just like with the strings, there are no quotes.
