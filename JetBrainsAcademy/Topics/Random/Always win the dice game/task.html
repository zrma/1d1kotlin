To show them how it works we'll play the six-sided dice game with them. The game is simple: they roll the dice N times, then we throw it N times, and whoever got a greater sum wins the bank. Our task is to hack the system and provide a seed for the generator that wins the game for a given integer number of steps N. Let's try different seeds until we find the one that guarantees our victory, and then return the seed.
