Is there a key?

You are given Map<String, String> Map and String s as input. Print true if the Map maps one or more keys to the s and the Map has a key s; otherwise, return false.


Sample Input:
{one to two}, {three to one}, {four to five}
one

Sample Output:
true


Sample Input:
{10 to 20}, {30 to 1}, {55 to 23}
55

Sample Output:
false
