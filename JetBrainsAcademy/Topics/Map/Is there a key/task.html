Is there a key?

The containsKeyAndValue function gets map: Map < String, String > and the string value: String as input parameters. If the map contains one or more keys and one or more values that are equal to value, the function returns true, otherwise returns false.


Sample Input:
{one to two}, {three to one}, {four to five}
one

Sample Output:
true


Sample Input:
{10 to 20}, {30 to 1}, {55 to 23}
55

Sample Output:
false
