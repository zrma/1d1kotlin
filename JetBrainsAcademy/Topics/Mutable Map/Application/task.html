Application

A beginner programmer is a site administrator. He has a map where for each user there is a key (the login) and a value (the password). Now he needs to add the data of a new user. Write a function addUser that takes a map with user data userMap, a login, and a password and adds the latter two to userMap.

If userMap already contains a user with such a login, leave userMap unchanged and print the following string: "User with this login is already registered!".

Input: userMap: Map<String, String>, login: String, password: String

Output: updated userMap: MutableMap<String, String>


Sample Input:
new_login
new_password

Sample Output:
{user1@mail.com=qwerty123, hi_john@mail.com=abcdef00, dr_mike@mail.com=000000, new_login=new_password}


Sample Input:
user1@mail.com
qwerty123

Sample Output:
User with this login is already registered!
{user1@mail.com=qwerty123, hi_john@mail.com=abcdef00, dr_mike@mail.com=000000}



Memory limit: 256 MB
Time limit: 8 seconds
