Concatenate 3 strings with separator

Write a function that concatenates three strings into one using a special separator. If the separator is not specified, use a single space.

The first three lines of the input contain three lines. The fourth line of the input contains a line separator. If there is no special separator, the line is NO SEPARATOR.

Output the result of the concatenating function.


Sample Input:
Hyperskill
Kotlin
Programming
~~~

Sample Output:
Hyperskill~~~Kotlin~~~Programming


Sample Input:
abc
def
ghi
NO SEPARATOR

Sample Output:
abc def ghi



Memory limit: 256 MB
Time limit: 8 seconds
