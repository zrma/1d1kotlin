Safe round

Here is a program that uses the round function according to this logic: if the input number is greater than 1000, it returns null, otherwise it returns the number. Do not change the round function: fix the output. It has to return 0 in null case.


Sample Input:
-5

Sample Output:
-5


Sample Input:
5000

Sample Output:
0
