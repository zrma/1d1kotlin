One after another

Write a program that:
1) if args size < 3, invokes main(args: Array<String>) with three input arguments: first, second, and third;

2) otherwise, prints the arguments – each on a new line.



Memory limit: 256 MB
Time limit: 8 seconds

Caution

You may see errors in your code or execution results due to missing context. Don’t worry about it, just write the solution and press Check.
