Square root

Now you know that Math groups multiple functions; one of them is Math.sqrt, that calculates the square root of passed value.

Call Math.sqrt function in the same way as we called the Math.abs, calculate the square root of the number and print the resulting value.


Sample Input:
4

Sample Output:
2.0


Sample Input:
16

Sample Output:
4.0



Memory limit: 256 MB
Time limit: 8 seconds
